# MIMIR  
  
## Project Context System  
  
Version:   
0.1.0  
  
Status:   
Experimental  
  
Purpose:   
Crear un sistema para preservar, reconstruir y actualizar el contexto de proyectos asistidos por IA.  
  
Core Principle:   
Las conversaciones son temporales. El conocimiento es permanente.  
  
Current Context:   
Este proyecto está en fase de diseño del estándar Mimir.  
  
Main Goal:   
Crear un sistema que permita a cualquier IA comprender un proyecto rápidamente sin depender del historial de conversaciones.
