# CONTEXT  
  
## Proyecto actual  
  
Mimir  
  
## Estado  
  
Diseño inicial.  
  
## Objetivo actual  
  
Crear el primer sistema de contexto persistente para trabajar con IA desde celular.  
  
## Qué estamos construyendo  
  
Un sistema que permita que una IA pueda retomar un proyecto leyendo información estructurada.  
  
## Decisiones importantes  
  
- Mimir no depende de una IA específica.  
- El conocimiento pertenece al proyecto.  
- Las conversaciones no son la fuente de verdad.  
- La información importante debe quedar registrada.  
  
## Próximo paso  
  
Diseñar el primer modelo de Context Snapshot.  
  
## Preguntas abiertas  
  
- ¿Qué información mínima necesita una IA para entender un proyecto?  
- ¿Cómo se actualiza el contexto sin trabajo manual?  
- ¿Cuál será la estructura definitiva?
